# Mi Primer PR

Este es un repositorio de práctica para aprender el flujo de trabajo de Pull Requests en GitHub.

## Sobre este proyecto

Este repositorio contien un ejemplo simple para practicar cómo:

1. Crear una rama
2. Hacer un cambio
3. Abrir un Pull Request
4. Revisar y fusionar (merge)

## Uso

Clona el repositorio y explora los archivos para empezar.
